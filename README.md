# teste-framework-frontend

Frontend do teste para framework in [react](https://reactjs.org/)

## Installation

Use the package manager [yarn](https://classic.yarnpkg.com/) to install.

## Install dpendencies

```bash
yarn
```

## Run application

```bash
yarn dev
#http://localhost:3000/
```
